# 用語集

## React

Web画面を作るための道具です。

画面を小さな部品に分けて作ります。

## コンポーネント

画面の部品です。

例：

- 商品カード
- ボタン
- 商品詳細
- 店舗管理画面

## TypeScript

JavaScriptに「データの形」を決める機能を足したものです。

例：

```ts
price: number
```

これは「priceは数字です」という意味です。

## Vite

Reactの開発環境を簡単に作るための道具です。

画面の起動や本番用のファイル作成をしてくれます。

## props

親の部品から子の部品へ渡すデータです。

例：

```text
ProductCardに商品データを渡す
```

## state

画面の中で変化するデータです。

例：

- 選択中の商品
- 商品の在庫
- 店舗ポイント
- 今表示している画面

## build

本番用にファイルを作ることです。

`npm run build` が失敗すると、本番でも動かない可能性があります。

## lint

コードの書き方に問題がないか確認することです。

## API

フロントとバックエンドがやり取りするための入口です。

例：

```text
商品一覧をください
購入します
```

こういうお願いを送ります。
