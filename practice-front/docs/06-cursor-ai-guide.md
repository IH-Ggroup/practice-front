# 06. CursorでAIを使う時の聞き方

CursorでAIを使うのはOKです。

ただし、丸投げすると分からないコードが増えます。次のように聞いてください。

## 良い聞き方

```text
src/components/ProductCard.tsxを読んで、
商品カードにポイント報酬を表示する方法を教えて。
できるだけ今のコードの形を崩さないで。
```

```text
このエラーの意味を初学者向けに説明して。
どのファイルのどこを直せばいいかも教えて。
```

```text
src/App.cssの.product-cardだけを見やすくしたい。
色は大きく変えず、余白と文字サイズだけ調整して。
```

## 悪い聞き方

```text
全部いい感じにして
```

```text
完璧なECサイトを作って
```

```text
全部書き直して
```

この聞き方だと、理解できないコードが増えやすいです。

## AIに頼んだ後に必ずやること

AIが出したコードを貼ったら、次を確認します。

- どのファイルが変わったか
- 何のための変更か
- 自分で説明できるか
- `npm run build` が通るか
- 画面が壊れていないか

## コピペ禁止ではない

コピペ自体は悪くありません。

ただし、貼ったコードを説明できないまま進むと、本番で直せなくなります。

## おすすめの使い方

まず自分で10分考える。

それでも分からなければAIに聞く。

AIの回答を読んで、必要な部分だけ使う。

最後に、変更内容を自分の言葉でメモする。
