# 03. React基礎タスク

このページは、Reactを少しずつ理解するための練習です。

## タスク1：商品データを増やす

開くファイル：

```text
src/data/sampleProducts.ts
```

やること：

- 商品を1つ追加する
- 商品名を変える
- 価格を変える
- 在庫を変える

確認すること：

- 商品一覧に追加した商品が出る
- 在庫0なら売り切れになる

## タスク2：商品カードの表示を変える

開くファイル：

```text
src/components/ProductCard.tsx
```

やること：

- 表示する文字を変える
- 価格の位置を変える
- 店舗名の見た目を変える

確認すること：

- 商品カードの表示が変わる
- エラーが出ない

## タスク3：購入ボタンの文章を変える

開くファイル：

```text
src/components/ProductDetail.tsx
```

やること：

- ボタンの文字を変える
- 売り切れ時の文字を変える
- 商品説明の下に注意文を追加する

確認すること：

- ボタンの文字が変わる
- 在庫0の時にボタンが押せない

## タスク4：商品出品フォームを触る

開くファイル：

```text
src/components/ProductCreateForm.tsx
```

やること：

- 入力欄の名前を変える
- エラーメッセージを変える
- 初期価格を変える

確認すること：

- 入力して商品を追加できる
- 商品一覧の先頭に追加される

## タスク5：店舗レベルの表示を変える

開くファイル：

```text
src/components/StoreDashboard.tsx
```

やること：

- 「店舗レベル」の表示を変える
- 次のレベルまでの文章を変える
- 商品一覧の表に表示する情報を増やす

確認すること：

- 購入後にポイントが増える
- レベルが変わる
- 店舗管理画面に反映される

## タスク6：自分で1つ機能を追加する

例：

- 商品カードに「ポイント報酬」を表示する
- 店舗管理画面に「出品数」を表示する
- 売り切れ商品を灰色っぽくする
- 価格が3000円以上の商品に「高額商品」と表示する

小さくてOKです。

大事なのは、追加した機能を自分で説明できることです。
