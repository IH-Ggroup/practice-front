# 02. 1週目のタスク

1週目は、難しい機能を作るよりも「Reactの画面を触れる状態」にすることが目標です。

## 今週のゴール

```text
Reactの画面が2人のPCで動く
↓
商品が表示される
↓
商品を選ぶと詳細が変わる
↓
購入ボタンを押すと在庫が減る
↓
在庫0なら売り切れになる
```

## すわのタスク

- [ ] 自分のPCで画面を起動する
- [ ] `src/App.css` を開いて、背景色を少し変えてみる
- [ ] 商品一覧のカードの余白を変えてみる
- [ ] 店舗管理画面の数字表示を見やすくする
- [ ] サイト全体のメインカラー案を2つ出す
- [ ] 金曜日に「どこを変えたか」を説明する

## 櫻井のタスク

- [ ] 自分のPCで画面を起動する
- [ ] `src/components/ProductCard.tsx` を読む
- [ ] 商品カードに表示する情報を1つ増やす
- [ ] `src/data/sampleProducts.ts` に商品を1つ追加する
- [ ] 商品詳細画面の説明文の見た目を調整する
- [ ] 金曜日に「どこを変えたか」を説明する

## 2人でやること

- [ ] 商品カードのデザインを決める
- [ ] ボタンの色を決める
- [ ] 店舗レベルの見せ方を決める
- [ ] 商品画像の表現をどうするか決める
- [ ] 画面を見て「分かりにくいところ」を3つメモする

## やらなくていいこと

今週は、以下はやらなくて大丈夫です。

- ログイン
- Spring Boot接続
- Minecraft接続
- スマホ対応
- すごいアニメーション
- 完璧なデザイン

## 金曜日に見せるもの

金曜日には、次をチームに見せてください。

- 動いている画面
- 変更したファイル
- 苦戦したところ
- 次に直したいところ

説明は完璧でなくて大丈夫です。自分の言葉で話せればOKです。
