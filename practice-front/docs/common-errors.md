# よくあるエラー

## `npm install` が失敗する

考えられる原因：

- Node.jsが入っていない
- ネットにつながっていない
- フォルダの場所が違う

確認するコマンド：

```bash
node -v
npm -v
pwd
```

## `npm run dev` が動かない

まず、今いる場所を確認します。

```bash
pwd
```

`practice-front` の中にいる必要があります。

## TypeScriptの赤いエラーが出る

よくある原因：

- 商品データに必要な項目が足りない
- 数字の場所に文字を入れている
- ファイル名や部品名を間違えている

例：

```text
Property 'price' is missing
```

これは「priceが足りません」という意味です。

## 画面が真っ白になる

ブラウザの開発者ツールを開いて、Consoleを見ます。

よくある原因：

- importのパスが間違っている
- 関数名を間違えている
- データがundefinedになっている

## 商品が追加されない

確認すること：

- 商品名を入力したか
- 価格が0以下になっていないか
- 在庫がマイナスになっていないか
- エラーメッセージが出ていないか

## Gitで怒られた

まずこれを見ます。

```bash
git status
```

分からなければ、エラー文をそのままチームに貼ってください。
