# AGENTS.md

このリポジトリは、MineTenantのフロント担当がReact + TypeScript + Viteを学ぶための練習用リポジトリです。

AIエージェントが作業する時は、次のルールを守ってください。

## 大事な方針

- 初学者が読む前提で、難しい書き方を避ける
- Reactの基本を学ぶことを優先する
- 便利すぎるライブラリを勝手に増やさない
- 本番用の複雑な設計にしすぎない
- 変更理由を短く分かりやすく説明する
- すわ・櫻井が自分で直せるコードにする

## 今は入れないもの

- Reduxなどの状態管理ライブラリ
- React Router
- UIコンポーネントライブラリ
- Tailwind CSS
- バックエンド接続
- 認証
- WebSocket
- Minecraft連携

これらは、基本の画面練習が終わってから追加します。

## コードの書き方

- 1つのファイルを大きくしすぎない
- 商品、店舗、フォームなど、意味のある単位でファイルを分ける
- TypeScriptの型を使う
- 変数名は意味が分かる名前にする
- コメントは本当に必要な場所だけにする
- エラー表示、売り切れ表示、空の状態を忘れない

## 確認すること

変更後は、最低限次を実行してください。

```bash
npm run build
npm run lint
```

画面を変えた場合は、可能なら `npm run dev` で表示も確認してください。
